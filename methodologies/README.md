# Méthodologies

Méthodes et frameworks d'investigation, indépendants de tout outil.

Cibles typiques :

- Boucle d'investigation (question → hypothèse → requête → pivot).
- Triage d'alerte : critères de qualification / déqualification.
- Timeline reconstruction, corrélation multi-sources.
- Pivots classiques (user → host → process → network → file).
- Frameworks externes utiles : MITRE ATT&CK, Kill Chain, Diamond Model — quand
  et comment les utiliser.
- Hypothèses concurrentes, biais de confirmation, qualité de preuve.

Une fiche décrit un **patron de raisonnement**, illustré par des exemples
abstraits, sans cas réel.
